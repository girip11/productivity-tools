# Hyper terminal

## Installation

- [Hyper installation](https://hyper.is/#installation)

```Bash
sudo dpkg -i "<path_to_debian_file>"
```

## Uninstall

```Bash
sudo dpkg -l | grep hyper
sudo dpkg -r hyper
sudo rm -fr ~/.hyper_plugins
sudo rm ~/.hyper.js
```
