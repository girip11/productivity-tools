# Capture terminal images

## [Carbon.now.sh][1]

- Website to screen capture terminal with code
- Custom theme selection
- Multiple language support
- can tweet, upload to imgur
- can export to PNG or SVG

## [asciinema][2]

- Record terminal
- cross platform software that requires [installation](https://asciinema.org/docs/getting-started).
- Locally recorded video can be uploaded to [asciinema.org][1] and the uploaded URL can then be shared with others.

[1]: https://carbon.now.sh/
[2]: https://asciinema.org/
